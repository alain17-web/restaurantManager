This page's purpose is to be the "homepage" of the Dashboard. Besides the DashboardSidebar and DashboardNavbar present on each page in the dashboard, it contains 3 Widget components aligned horizontally that display a notification each time a new order / purchase / booking is made. In the center of the page, the current time ( TimeDisplay component ) and date ( DateDisplay component ) are displayed above the RosterCard component that shows the avatars and first name of the employees on duty.
